Nuestro juego se centra en Stitch, un personaje principal de color azul que el jugador controlará dentro de un espacio limitado.
Su objetivo es recolectar estrellas, las cuales le permitirán desbloquear una pistola. Estas armas se recargan mediante cristales de 
color verde neón que el jugador deberá encontrar a lo largo de la partida.

El propósito es eliminar a Leroy, un personaje similar a Stitch pero de color rojo, cuya función es actuar como obstáculo. Cada vez que
Stitch entre en contacto con Leroy, el juego se reiniciará.

El jugador gana cuando logre eliminar a todos los Leroy y recolecte todas las estrellas.
